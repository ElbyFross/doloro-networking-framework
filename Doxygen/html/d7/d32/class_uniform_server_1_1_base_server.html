Detailed Description

Class that provide base server features and envirounment static API.

Part of class that provide methods for establishing transmisssion via PipesProvider.

Part of class that provide methods can be started as thread for server init;

◆ SendAnswerViaPP()

static bool UniformServer.BaseServer.SendAnswerViaPP ( string  answer,
UniformQueries.QueryPart []  entryQueryParts 
)
static

Open server line using PipesProvider that will send answer backward to cliend by dirrect line. Line will established relative to the data shared by client query.

Using this method you frovide uniform revers connection and not need to create a transmission line by yourself.

Recommended to use this methos by default dor duplex connection between sever and clients.

Parameters
serverInstance of server that will provide multithreading implementation.
answerMessage that will sent by server to target client.
entryQueryPartsParts of query that was recived from client. Method will detect core part and establish backward connection.
